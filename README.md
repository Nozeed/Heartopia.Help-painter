# üçé Heartopia Help Painter ‚Äî macOS Installation Guide

## ‚úÖ Requirements

Before starting, make sure you have:

* macOS 11+ (Big Sur or newer recommended)
* Internet connection
* Terminal access
* Git installed
* Python 3.10 or newer

## 1Ô∏è‚É£ Install Homebrew (if not installed)

Homebrew is the recommended package manager for macOS.

Open **Terminal** and run:

```bash

```

Verify installation:

```bash

```

## 2Ô∏è‚É£ Install Python

Install Python using Homebrew:


Check Python version:

```bash

```

## 3Ô∏è‚É£ Install Git (if needed)

```bash

```

Verify:

```bash

```

---

## 4Ô∏è‚É£ Clone the Repository


## 5Ô∏è‚É£ Create Virtual Environment (Recommended)

Create a Python virtual environment:


```bash

```

You should now see `(venv)` in your terminal.

## 6Ô∏è‚É£ Install Dependencies

Install required packages:


## 7Ô∏è‚É£ Run the Application

Start the app using:
