# ขั้นตอนการติดตั้งและรัน (Full Commands)
1 ติดตั้ง python 3+ ก่อน (https://www.python.org/downloads/windows/)

2 แตกไฟล์ Heartopia-Image-Painter.zip ไว้ที่ desktop

3 ใช้ PowerShell (Run as administrator) ในการรัน

4 ปลดล็อกสิทธิ์การรัน Script (ทำครั้งเดียว)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

5 เข้าแฟ้มของโปรเจค แฟ้มต้องชื่อ Heartopia.Help-painter-main และต้องอยู่ที่ Desktop
cd "$HOME\Desktop\Heartopia.Help-painter-main"

6 สร้าง Virtual Environment ():
python -m venv .venv

7 ทำการ Activate
.\.venv\Scripts\Activate.ps1

8 ติดตั้ง Library ที่จำเป็น (สำคัญมาก)
pip install -r requirements.txt

9 เริ่มใช้งานโปรแกรม
python main.py
