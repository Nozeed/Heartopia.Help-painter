# ขั้นตอนการติดตั้งและรัน (Windows)
1 ติดตั้ง python 3+ ก่อน
https://www.python.org/ftp/python/3.14.3/python-3.14.3-amd64.exe

2 แตกไฟล์ Heartopia-Image-Painter.zip ไว้ที่ desktop

3 ใช้ PowerShell (Run as administrator) ในการรัน
Start Menu → พิมพ์ PowerShell → Run as Administrator

4 ปลดล็อกสิทธิ์การรัน Script (ทำครั้งเดียว)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

5 เข้าไปยังโฟลเดอร์โปรเจกต์ แฟ้มต้องชื่อ Heartopia.Help-painter-main และต้องอยู่ที่ Desktop
cd "$HOME\Desktop\Heartopia.Help-painter-main"

6 สร้าง Virtual Environment ():
python -m venv .venv

7 ทำการ Activate
.\.venv\Scripts\Activate.ps1

8 ติดตั้ง Library ที่จำเป็น (สำคัญมาก)
pip install -r requirements.txt

9 เริ่มใช้งานโปรแกรม
python main.py

10 ปิด Virtual Environment
เมื่อไม่ได้ใช้งานโปรแกรมแล้วให้รันปิด :
deactivate

การกลับมารันโปรแกรมอีกครั้งหลังปิดเครื่อง (Windows PowerShell)
1️⃣ เปิด PowerShell (ขั้นตอนติดตั้ง 3)
2️⃣ เข้าไปยังโฟลเดอร์โปรเจกต์ (ขั้นตอนติดตั้ง 5)
3️⃣ เปิดใช้งาน Virtual Environment (ขั้นตอนติดตั้ง 6)
4️⃣ รันโปรแกรม  (ขั้นตอนติดตั้ง 9)
สรุปคือ 3 คำสั่ง หลังเปิด PowerShell:
- cd "$HOME\Desktop\Heartopia.Help-painter-main"
- .\.venv\Scripts\Activate.ps1
- python main.py
