# ขั้นตอนการติดตั้งและรัน (Full Commands)

- ติดตั้ง python 3+ ก่อน (https://www.python.org/downloads/windows/)

- แตกไฟล์ Heartopia-Image-Painter.zip ไว้ที่ desktop

- ใช้ PowerShell (Run as administrator) ในการรัน

- ปลดล็อกสิทธิ์การรัน Script (ทำครั้งเดียว)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

- เข้าแฟ้มของโปรเจค แฟ้มต้องชื่อ Heartopia.Help-painter-main และต้องอยู่ที่ Desktop
cd "$HOME\Desktop\Heartopia.Help-painter-main"

- สร้าง Virtual Environment ():
python -m venv .venv

- การ Activate
.\.venv\Scripts\Activate.ps1

- ติดตั้ง Library ที่จำเป็น (สำคัญมาก)
pip install -r requirements.txt

- เริ่มใช้งานโปรแกรม
python main.py

- วิดีโอสอนใช้งาน
จากนั้นดูคลิปวิธีตั้งค่าโปรแกรม https://youtu.be/nr_-wqowNoM
วิธีแก้สีเพี้ยนวาดไม่ตรง https://youtu.be/7ReYhclehEA
