ติดตั้ง python 3+ ก่อน (https://www.python.org/downloads/windows/)
แตกไฟล์ Heartopia-Image-Painter ไว้ที่ desktop

ใช้ PowerShell ในการรัน

ปลดล็อกสิทธิ์การรัน Script (ทำครั้งเดียว)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

ขั้นตอนการติดตั้งและรัน (Full Commands)
cd "$HOME\Desktop\Heartopia-Image-Painter"

สร้าง Virtual Environment ():
python -m venv .venv

การ Activate
.\.venv\Scripts\Activate.ps1


ติดตั้ง Library ที่จำเป็น (สำคัญมาก)
pip install -r requirements.txt

เริ่มใช้งานโปรแกรม: (หากกลับมาใช้โปรแกรมอีกครั้งให้รันตรงนี้เลย)
python main.py

จากนั้นดูวิธีตั้งค่าโปรแกรมตามคลิปที่สอนได้เลย
